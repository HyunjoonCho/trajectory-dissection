# trajectory-dissection
2025F CS632 Term Project &lt;Dissecting LLM Agent Performance:  A Cross-Model and Cross-Task Analysis>

### References
- Bouzenia et al. "Understanding Software Engineering Agents: A Study of Thought-Action-Result Trajectories." ASE 2025
- Ceka et al. "Understanding Software Engineering Agents Through the Lens of Traceability: An Empirical Study." arXiv 2025

We apply established trajectory analysis techniques along with simpler variants to understand agent behavior. A key objective is to determine if these metrics can reliably predict task success without requiring manual inspection.
